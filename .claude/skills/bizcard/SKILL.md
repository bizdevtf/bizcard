# 명함 → vCard(.vcf) 변환

## 처리 절차

1. **이미지 읽기**: 사용자가 보낸 명함 사진을 Read 도구로 읽는다. 여러 장이면 각각 처리한다.
2. **정보 추출**: 명함에서 다음 항목을 추출한다 — 이름(필수), 회사, 부서, 직함, 휴대폰, 유선전화, 팩스, 이메일, 주소, 웹사이트.
   - 한글/영문이 병기된 명함이면 **한글 이름을 기본**으로 쓰고, 영문 이름은 note에 넣는다.
   - 전화번호는 하이픈 포함 형식으로 정규화한다 (예: 010-1234-5678, 02-123-4567).
   - 읽히지 않거나 불확실한 항목은 추측하지 말고 비워 둔다.
3. **JSON 작성**: 추출 결과를 스크래치패드에 JSON 파일로 저장한다. 형식은 `scripts/make-vcard.mjs` 상단 주석 참조.
4. **vCard 생성**: `node scripts/make-vcard.mjs <input.json> <output.vcf>` 실행 (저장소 루트에서). 파일명은 `이름_회사.vcf` 형태로 한다.
5. **전송**: SendUserFile로 .vcf 파일을 보내면서, 추출한 정보를 표로 요약해 보여주고 "휴대폰에서 파일을 열면 연락처에 저장됩니다"라고 안내한다.
6. **확인 요청 없이 바로 처리한다.** 다만 이름이나 전화번호를 읽을 수 없으면 파일 생성 전에 사용자에게 물어본다.

## 주의

- 명함 여러 장이 한 사진에 있으면 각 명함마다 별도 vCard를 만들어 하나의 .vcf 파일에 이어 붙인다(스크립트가 JSON 배열 입력을 지원).
- 추출 정보는 개인정보다. 저장소에 커밋하지 말고 스크래치패드에서만 다룬다.
