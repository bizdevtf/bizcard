# bizcard — 명함 → 연락처 자동 저장

명함 사진을 찍어 채팅(Claude)으로 보내면, Claude가 명함 내용을 인식해
휴대폰 연락처에 신규 저장해 주는 기능을 개발하는 프로젝트입니다.

## 목표 흐름

1. 휴대폰으로 명함 사진 촬영
2. 사진을 채팅 형태로 Claude에 전송
3. Claude가 이미지에서 이름·회사·직함·전화번호·이메일 등 추출
4. 추출된 정보를 휴대폰 연락처에 신규 등록
